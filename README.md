# ✨ Animation Collection

Bộ sưu tập các hiệu ứng CSS animation — tái sử dụng được, không cần framework.

Mỗi hiệu ứng là một folder độc lập với file `index.html` có thể mở thẳng trên trình duyệt.

---

## Danh mục

### 📁 [card/](./card/)
Hiệu ứng liên quan đến thẻ bài viết / card.

| Hiệu ứng | Mô tả | Demo |
|---|---|---|
| [hover-card-blur-effect](./card/hover-card-blur-effect/) | Hover vào card nào thì card đó hiện rõ, các card khác tự blur mờ | [index.html](./card/hover-card-blur-effect/index.html) |

---

## Cách dùng

```
1. Clone repo về máy
2. Mở folder hiệu ứng muốn xem
3. Mở file index.html trên trình duyệt
```
